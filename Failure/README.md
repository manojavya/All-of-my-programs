# My CLR Project

This project is a simple CLR application targeting the .NET Framework.

## Overview

This application serves as a basic template for developing CLR applications. It includes the necessary files and structure to get started with .NET Framework development.

## Project Structure

- `src/Program.cs`: Contains the entry point of the application with the `Main` method.
- `my-clr-project.csproj`: The project file that includes settings and configurations for building the project.
- `README.md`: Documentation for the project.

## Usage

To build and run the application, use the following commands:

1. Open a terminal and navigate to the project directory.
2. Build the project:
   ```
   dotnet build
   ```
3. Run the application:
   ```
   dotnet run --project src/Program.cs
   ```
